CSV Format

Your CSV file should have the following format (one transaction per line):

payer,debtor,amount

Example:

Alice,Bob,25.50
Bob,Charlie,15.00
Charlie,Alice,30.75

SplitUp will analyze all transactions and show you the minimum number of payments needed to settle all debts.
